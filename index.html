Projects

Exact and Matheuristic algorithms for TSP - Operations Research 2

The purpose of this project is to illustrate various techniques used to approach the Travelling Salesman Problem (TSP) and to compare the results obtained by testing the aforementioned methods with respect to reference instances.
Here you can find our report.

March 2017 - September 2018

Clustering of Wikipedia Pages - Data Mining

The goal of this project was to try to assess to what extent a clustering is consistent with the categories attached to the Wikipedia pages, and to find out which preprocessing, document representation and clustering type, yields the best results.
We used a sample of approximately 100k WikiPages and implemented severals clustering algorithms and performance metrics.
Here you can find our results.

April 2017 - June 2017

Pose Recognition algorithm - Autonomous Robotics

The purpose of this project is to collect a dataset after an appropriate configuration of a sensor network and to measure the quality of the results provided by an algorithm of pose recognition. Afterwards we tried to improve the pose recognition algorithm initially used by implementing different techniques, for example using a multiframe recognition approach.
Here you can find our report.

December 2017 - June 2018

Ranked Keyword Searches on Graphs - Database Management Systems

This project was aimed at trying to reproduce some literature results about keyword searches on Graphs, shortyl a reproducibility project.
In particular we implemented BLINKS, a bi-level indexing algorithm, and used imdb and mondial datasets for the testing phase.
Click here for the original paper and here for our results.

June 2016 - January 2017

DASH Streaming - Computer Networks

This is a simple DEMO of a client-server DASH video streaming. We have done a comparison of 3 different algorithms:

  • BitRate Based, our own implementation.
  • Buffer Based, our own implementation
  • Festive, taken from J. Jiang, V. Sekar, H. Zhang work. Here you can find the original paper
For the results see our report.
